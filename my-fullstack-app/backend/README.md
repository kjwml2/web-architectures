# Backend API Documentation

This is the backend part of the fullstack application built with Express and TypeScript.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Node.js (v14 or later)
- npm (v6 or later)

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/my-fullstack-app.git
   ```
2. Navigate to the backend directory
   ```bash
   cd my-fullstack-app/backend
   ```
3. Install the dependencies
   ```bash
   npm install
   ```

### Running the Application

To start the server, run the following command:

```bash
npm start
```

The server will start on `http://localhost:3000`.

### API Endpoints

- **GET /health**: Returns a simple health check response.

### Error Handling

The application includes error handling middleware that captures errors and sends appropriate responses.

### Logging

The application uses a simple logging utility to log messages to the console.
